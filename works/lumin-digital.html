Lumin Digital

Delightful Animations

2022 | Motion | Motion Design | lumindigital.com ↗

Introduction

Lumin Digital is a fintech company specializing in cloud native digital banking solutions. Their site needed several animations to support the user onboarding process of their banking platform.


Motion - UX/UI

I. Objectives

Lumin needed animations to help aid their user onboarding process. Things like: requesting a new credit card, signing up for online statements, and security settings. Some of the animations had already been created, so there were existing styles and motion to align to. Another constraint was that the animations were limited to 4 brand colors (and some additional gray values) that would later be targeted for customization. 



II. Challenges

The most challenging part of this project was conceptualizing the complex ideas involved in a banking app with simple, easy to understand animations. Creating simple illustrations that connected to these broader ideas and matched what previously existed yet still conveyed the concepts to the user.


III. Cast

To keep the animations consistent, basic characters were created to be reused throughout the series where possible. We designed main characters such as a house, safe, wallet and certificate as well as things like check marks and plus sign with the intent of being distributed throughout the animations as a way to create a familiarity amongst them.



IV. Colors

A color token system was created for consistency across the illustrations and assigned to the main colors. We set default token values in the design, used those to name the AE layers, and exported them as JSON files. This allowed developers to target and replace default values with brand-specific colors tailored to any client.



V. Delivery

We delivered the finished product by exporting json files using bodymovin. We were able to meet the brief and overcame the challenges by utilizing the brand colors, illustration styles and harnessing power of Lottie. This allows for playback of complex animations natively in various applications without the need for heavy video files.


Conclusion

Ultimately, integrating motion graphics into Lumin's user onboarding process involved simplifying complex banking concepts into clear animations with a limited color palette. The project highlighted key user actions, such as requesting a new credit card, using familiar patterns for enhanced understanding. Visual communication of concepts was achieved through strategic characters, contributing to a cohesive overall narrative. A token system streamlined customization for developers, allowing easy updates, and Lottie facilitated seamless web integration.
